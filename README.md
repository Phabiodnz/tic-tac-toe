# Tic-Tac-Toe

Simple Tic-Tac-Toe game built with HTML, CSS and JavaScript using a factory function to manage game state.

## Features
- Dynamic board rendering  
- Player turn switching  
- Win + draw detection  
- Board lock after game ends  
- Play and Reset buttons  
- No global game state

## Files
index.html
style.css
script.js

## How to Run
Just open index.html in the browser.

## Structure
- createBoard() → creates and manages the board  
- checkWin() → checks win conditions  
- initGame() → initializes events and game instance
